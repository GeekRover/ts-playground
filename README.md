# TypeScript Playground 🚀

> A personal learning space to practice and master TypeScript concepts.

## Overview

Welcome to my TypeScript playground! This repository is a dedicated space where I explore and practice TypeScript features, patterns, and best practices. Whether it's diving into types, generics, interfaces, or advanced language features, everything is documented and organized for easy reference.

## 📚 What's Inside

- **Type System Exercises**: Exploring interfaces, types, unions, intersections, and type guards
- **OOP Concepts**: Classes, inheritance, access modifiers, and design patterns
- **Generics & Advanced Types**: Generic functions, generic classes, conditional types, and mapped types
- **Async Patterns**: Promises, async/await, callbacks, and error handling
- **Practical Examples**: Real-world implementations and use cases
- **Best Practices**: Code organization, linting, and testing approaches

## 🛠️ Tech Stack

- **Language**: TypeScript
- **Runtime**: Node.js
- **Configuration**: [tsconfig.json](tsconfig.json)

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/GeekRover/ts-playground.git
cd ts-playground
npm install -g typescript
```

### Running Examples

To compile and run TypeScript files:

```bash
# Compile TypeScript
tsc

# Or compile a specific file
tsc src/basics.ts
```

## 📝 Learning Path

This playground is structured as a progressive learning journey:

1. **Basics** - Fundamental types and syntax
2. **Intermediate** - More complex patterns and advanced types
3. **Advanced** - Generics, conditional types, and metaprogramming

Each module builds on concepts from previous ones, creating a comprehensive TypeScript foundation.

## 💡 Tips for Learning

- **Experiment Often**: Don't just read, write code and test it
